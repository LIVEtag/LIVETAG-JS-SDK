# Livetag Docs

## Requirements
- Node.js ~14.0
- npm ~6.14

## Setup
```bash
npm ci
```

## Build
```bash
npm run build
```

## Development
```bash
npm run dev
```

## Configuration
Create `.env` on root level of `/docs` if it does not exist:
```bash
cd ..
cp .env.example .env
```
and fill all variables.
